# HRCCA_reproducibility
This repository comprises code and analyses for the Human RPE and Choroid Cell Atlas (HRCCA) project.

Single-cell/nucleus quality control (ambient RNA correction, hard filtering, and doublet removal) was performed using the cellqc pipeline (https://github.com/lijinbio/cellqc). 

Data preprocessing, batch correction, and multi-sample integration followed the wrapper scripts published at https://github.com/RCHENLAB/HRCA_reproducibility/tree/main/utility/scrna*.
